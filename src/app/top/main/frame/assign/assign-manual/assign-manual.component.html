1. 새 과제 생성하기

새로운 과제 그룹을 생성합니다. 각 과제는 임의의 내담자에게 할당될 수 있으며, 한 내담자는 하나의 과제 그룹에 소속될 수 있습니다. 내담자가 특정 과제 그룹에 소속되면, 내담자는 해당 과제 그룹에서 제공하는 교육, 설문지, 과제를 수행하게 됩니다. DMHC는 기본적으로 2개의 과제 그룹 (자서전적 기억 훈련, 행동활성화)을 제공하며, 그 외에 내담자에게 필요한 과제를 치료자가 직접 커스터마이징 할 수 있습니다. 하나의 과제 그룹은 [교육], [설문지], [과제]로 구성됩니다.

1. [전체 과제 정보보기] 클릭

2. 과제 그룹 이름 입력

3. [내담자 교육 설정] 에서 생성한 교육을 선택합니다. 해당 교육은 내담자가 과제를 처음 시작할 때 보여지게 됩니다.

4. [이 그룹에 과제 추가] 버튼을 눌러 과제를 추가합니다.

2. 이 그룹에 과제 추가

한 과제 그룹에 과제는 복수로 존재할 수 있습니다. 과제 추가 버튼을 눌러 과제를 추가할 수 있습니다.

1. [이 그룹에 과제 추가] 클릭 한 그룹에 과제는 복수로 존재할 수 있으며, 버튼을 눌러 추가할 수 있습니다.

2. [과제명] 입력

3. [과제 지시문]을 입력합니다 과제 지시문은 내담자에게 과제가 주어질 때 [푸시 알림]으로 내담자에게 노출됩니다.

4. [세부 설정] 버튼을 눌러 해당 과제에 대한 세부 설정을 시행합니다.

3. 과제 세부 설정

과제에 대한 세부 설정 화면입니다. 과제 세부 설정은 [과제 생성 조건], [과제 생성 수량 선택], [내담자 입력방식 선택], [과제 푸시 알림 설정], [제시 과제 소멸 설정], [과제 제시 중지 설정] 등을 설정하게 됩니다.

1. 해당 버튼을 눌러 [과제 생성 조건] 세부사항을 설정합니다.

[과제 생성 조건]은 과제가 처음으로 생성되기 위한 조건이며,

1)내담자 첫 로그인 시 바로 해당 과제 생성
2)지정된 날짜 이후로 해당 과제 생성
3)내담자의 전체 과제 수행이 일정 횟수 이상 되었을 때 생성
4)내담자의 특정 과제 수행이 일정 횟수 이상 되었을 때 생성

등으로 구성되어 있습니다.

2. 해당 버튼을 눌러 [과제 생성 수량] 세부사항을 설정합니다.

[과제 생성 수량]은 반복해서 생성되는 과제의 양을 설정하는 조건이며

1)매일 지정된 개수 생성
2)특정 날짜마다 지정된 개수 생성
3)특정 요일마다 지정된 개수 생성
등으로 구성되어 있습니다.

3. 과제 세부 설정

과제에 대한 세부 설정 화면입니다. 과제 세부 설정은 [과제 생성 조건], [과제 생성 수량 선택], [내담자 입력방식 선택], [과제 푸시 알림 설정], [제시 과제 소멸 설정], [과제 제시 중지 설정] 등을 설정하게 됩니다.

1. 해당 버튼을 눌러 [내담자 입력방식] 세부사항을 설정합니다.

[내담자 입력방식]은 내담자가 과제를 수행하는 방식에 대한 설정이며, 과제 설정의 핵심입니다.

1)단순 텍스트 입력은 내담자가 과제를 자유롭게 주관식 텍스트 입력으로 수행해야 할 때 선택합니다. +) 사진 입력 받기를 체크하여 핸드폰에 저장된 사진을 업로드 할 수 있습니다.
2)일일 활동 기록지는 행동활성화에서 주로 사용되며, 하루의 활동을 내담자가 직접 기록하는 표 입니다.
3)주간 목표 계획지는 행동활성화에서 주로 사용되며, 일주일간 활동 목표를 계획하는 표 입니다.
4)주간 활동 기록지는 행동활성화에서 주로 사용되며, 일주일 동안의 목표 달성 여부를 평가하는 표 입니다.
5)설문지 작성을 선택한 경우 [설문지 설정]에서 미리 작성된 설문지를 내담자가 수행하게 됩니다.
6)사용자 정의 테이블 형식의 경우 표 형식의 입력창을 치료자가 직접 설계할 수 있습니다.
7)교육 다시보기의 경우, 내담자가 과제 그룹에 설정되어 있는 [심리교육]을 다시 보게 됩니다

3. 과제 세부 설정

과제에 대한 세부 설정 화면입니다. 과제 세부 설정은 [과제 생성 조건], [과제 생성 수량 선택], [내담자 입력방식 선택], [과제 푸시 알림 설정], [제시 과제 소멸 설정], [과제 제시 중지 설정] 등을 설정하게 됩니다.

1. 해당 란에서 [제시된 과제의 소멸]과 관련된 설정을 할 수 있습니다.

제시된 과제의 소멸 설정에서는, 주어진 과제를 일정 기간 혹은 일정 개수 이상 완료하지 않았을 때, 해당 과제를 삭제하는 조건을 설정합니다.

2. 해당 란에서 [과제 제시 중지]와 관련된 설정을 할 수 있습니다.

과제 제시 중지 설정에서는, 반복 제시되고 있는 과제를 영구적으로 중지하는 조건을 설정합니다.

4. Hash Tag 설정하기

Hash Tag는 반복해서 제시되는 과제의 제시문에 무작위성을 부여해야 할 때 사용됩니다. Hash Tag를 사용해서 반복해서 제시되는 과제이지만, 매번 다른 과제를 내담자에게 부여할 수 있게 됩니다.

ex) ‘#동물 단어를 보고 떠오르는 생각을 적어보세요’ 라는 지시문이 있고,
#동물 Hash Tag는 (강아지, 고양이) 일 때,
내담자에게 보여지는 지시문은
‘강아지 단어를 보고 떠오르는 생각을 보여주세요’ 혹은
‘고양이 단어를 보고 떠오르는 생각을 보여주세요’ 입니다 (무작위).
Hash Tag와 Hash Tag에 속하는 단어들은 자유롭게 설정할 수 있습니다.

1. [Hash Tag 설정하기] 클릭

2. Hash Tag 추가하기 버튼을 눌러 새로운 Hash Tag를 생성합니다. Hash Tag는 #를 앞에 붙여야 합니다

3. 추가/삭제 버튼을 눌러 해당 Hash Tag에 속하는 단어들을 추가하거나 삭제할 수 있습니다.

4. Hash Tag 그룹에 이미 작성된 Hash Tag 그룹을 하위 그룹으로 속하게 할 수 있습니다.
